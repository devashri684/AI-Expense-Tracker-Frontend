# 💰 AI Expense Tracker (Frontend)

This is the frontend of my AI Expense Tracker project built using React and Vite.

## 🚀 Features
- Add and manage expenses
- View expense list
- Responsive UI
- Connected with Spring Boot backend APIs

## 🛠️ Tech Stack
- React.js
- Vite
- Axios
- CSS
